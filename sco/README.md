# scope-sco

`scope-sco` is the spin-crossover add-on for SCOPE. It extends the core `scope` package with utilities for creating and analysing spin-crossover systems.

## Installation

Install the core package first, then install the add-on:

```bash
pip install scope
pip install scope-sco
```

For a local editable installation from this repository:

```bash
pip install -e core
pip install -e sco
```

## Command Line Interface

The package provides the `scope_sco` command. For help:

```bash
scope_sco -h
```
